# User Guide

## Getting Started

1. Launch: `python -m autonomiclab`
2. Click "Select Dataset Folder"
3. Choose a Finapres data folder
4. View markers in the left panel
5. Run analyses

## Analyses

### Valsalva Maneuver
Tests sympathetic nervous system response

### Stand Test  
Tests orthostatic blood pressure regulation

### Deep Breathing
Tests parasympathetic function via RSA

## Keyboard Shortcuts
- Ctrl+O: Open dataset
- Ctrl+S: Save results
- Ctrl+Q: Quit
